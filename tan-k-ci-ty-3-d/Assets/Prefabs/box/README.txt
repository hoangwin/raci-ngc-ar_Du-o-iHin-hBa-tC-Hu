Installing instructions:

Windows:
	XP/Vista:
		Type in "%appdata%\.minecraft\texturepacks"
			
	Windows 7:
		Type "C:\Users\*YourName*\AppData\Roaming\.minecraft\texturepacks"


Mac:
	Navigate to Users/*YourName*/Library/Application Support/Minecraft/texturepacks

Linux:
	Go to view options
	Check the 'show hidden folders' box
	Go to your home directory
	Open '.minecraft'
	Open 'texturepacks'

move the file into there
Done!
